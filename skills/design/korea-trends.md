# 2024-2025 한국 웹 UI/UX 트렌드

## 개요

한국 웹 디자인은 글로벌 트렌드를 빠르게 수용하면서도 독자적인 특성을 보입니다.
이 문서는 현재 한국 시장에서 주목받는 UI/UX 트렌드를 정리합니다.

> ⚠️ 트렌드는 빠르게 변하므로, 프로젝트 시작 시 반드시 최신 트렌드를 웹 검색으로 확인할 것

---

## 레이아웃 트렌드

### 1. 풀스크린 히어로
```
┌─────────────────────────────────────┐
│                                     │
│        100vh 배경 이미지/영상        │
│                                     │
│          대형 타이틀                 │
│          서브카피                    │
│          [CTA 버튼]                  │
│                                     │
└─────────────────────────────────────┘
```
- 첫 화면에서 강렬한 인상
- 영상 또는 고품질 이미지
- 스크롤 유도 애니메이션

### 2. 벤토 그리드 (Bento Grid)
```
┌──────────┬───────┬───────┐
│          │       │       │
│   2x2    │  1x1  │  1x1  │
│          │       │       │
├──────────┼───────┴───────┤
│   1x1    │      2x1      │
└──────────┴───────────────┘
```
- 불규칙한 그리드 레이아웃
- 다양한 콘텐츠 타입 조합
- 시각적 흥미 유발

### 3. 스크롤 스토리텔링
- 섹션별 순차적 스토리 전개
- 스크롤 연동 애니메이션
- 진행률 표시 (선택)

### 4. 오버사이즈 타이포그래피
- 대형 한글 타이틀 (60px+)
- 굵은 웨이트 사용
- 타이틀 자체가 비주얼 요소

## 컬러 트렌드

### 소프트 파스텔 베이스
```
연한 배경     + 비비드 포인트
#F8FAFC       #3B82F6
#FFF7ED       #F97316
#F0FDF4       #22C55E
```

### 그라데이션 활용
```css
/* 미묘한 배경 그라데이션 */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* 글래스모피즘과 결합 */
background: rgba(255, 255, 255, 0.25);
backdrop-filter: blur(10px);
```

### 다크 모드 기본 지원
- 라이트/다크 토글 제공
- 시스템 설정 자동 감지
- 부드러운 전환 애니메이션

## 인터랙션 트렌드

### 1. 마이크로 호버 효과
```css
/* 카드 호버 */
.card {
  transition: transform 0.3s ease, box-shadow 0.3s ease;
}
.card:hover {
  transform: translateY(-4px);
  box-shadow: 0 20px 40px rgba(0,0,0,0.1);
}

/* 버튼 호버 */
.button:hover {
  transform: scale(1.02);
}
```

### 2. 스크롤 연동 패럴랙스
- 배경/전경 다른 속도로 이동
- 요소별 순차 등장
- String Tune으로 구현

### 3. 부드러운 페이지 전환
- Next.js App Router 전환 애니메이션
- 콘텐츠 페이드 인/아웃
- 로딩 스켈레톤

### 4. 커스텀 커서 (선택)
```css
/* 링크 호버 시 커서 변경 */
a:hover {
  cursor: url('/cursor-pointer.svg'), pointer;
}
```

## 업종별 특화 트렌드

### 의료/뷰티
- 청결감 있는 화이트 베이스
- 신뢰감 주는 블루/그린 포인트
- 전문성 강조 타이포그래피
- **전후 비교 슬라이더**
- 의료진 소개 섹션

### 카페/레스토랑
- 따뜻한 톤 (베이지, 브라운)
- 음식 사진 풀스크린
- **메뉴 탭 네비게이션**
- 위치/영업시간 강조
- 인스타그램 연동

### 피트니스
- 다이내믹한 이미지
- 강렬한 컬러 (블랙, 레드, 옐로우)
- **프로그램 비교 테이블**
- 회원 후기 강조
- 무료체험 CTA

### 교육
- 신뢰감 있는 블루 계열
- **커리큘럼 타임라인**
- 수강 후기/합격 사례
- 상담 예약 강조
